# Ask Lenox — Vercel Ready

This is a clean Next.js deployment package for the Hillside at Lenox resident assistant.

## Locked rules included
- Front parking is limited to **30 minutes maximum**
- Monthly dues are late after the **10th of each month**
- Formal enforcement and account-specific issues are routed to property management / Community Liaison
- Emergencies route to 911 first

## Deploy on Vercel
1. Create a GitHub repository called `ask-lenox`
2. Upload all files from this folder to the repository
3. Go to Vercel
4. Add New → Project
5. Import the GitHub repo
6. Framework Preset should auto-detect as **Next.js**
7. Build Command: `next build`
8. Output Directory: leave blank
9. Click Deploy

## Add to Hillside website
In HOA Express:
1. Go to Pages
2. Add a new Link Page
3. Name it `Ask Lenox`
4. Paste the Vercel URL
5. Save and add to the main menu
